# Customer Sales Analysis

## Project Overview

This project analyzes customer and sales data using Python and Pandas. The goal is to understand customer behavior, sales performance, and identify key business insights.

## Objectives

-Load and explore datasets
-Perform data cleaning
-Analyze customer and sales data
-Create aggregations and pivot tables
-Visualize data using charts

## Technologies Used

-Python
-Pandas
-Matplotlib
-Jupyter Notebook

## Files Included

-customer_analysis.ipynb
-sales_data.csv
-customer_churn.csv
-analysis_report.pdf
-requirements.txt

## Analysis Performed

-Total Revenue Calculation
-Average Sales Calculation
-Highest Sale Identification
-Product-wise Sales Analysis
-Pivot Table Creation

## Visualizations

-Bar chart
-Pie Chart

## Installation

pip install -r requirements.txt

## Run Project

jupyter notebook

Open customer_analysis.ipynb and run all cells.

## Conclusion

The project successfully analyzes customer sales data and provides useful insights through data analysis and visualizations.
